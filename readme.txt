How to deply the project:


5Minuti-store-frontend:
edit the API_URL variable in javaScript.js so that it reflects the domain you want the users to reach
set up a web server using apache, ngnix or a similar program.
set the public_html folder as the root folder for the web server

5Minuti-store-backend:
set up a database using mysql or similar software
open application.properties.template and create your application.properties file with the settings that are correct for your setup
use maven to get a runnable jar file from the spring boot project
run the jar file
